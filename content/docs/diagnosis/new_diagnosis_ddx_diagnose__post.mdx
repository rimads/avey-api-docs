Generates a comprehensive clinical diagnosis and patient recommendations based on a patient's collected information. This endpoint takes a complete patient profile and returns a set of potential diagnoses and care recommendations.

### Query parameters can be used to toggle specific features:

* Differential Diagnosis
* Urgency (Triage)
* Specialist Recommendation
* Suggested Health Management Advice

**Note**: This endpoint is the final step in a diagnositc session after a null question is returned. If a full diagnostic session is too long for your use case, you can immediately receive patient recommendations by calling this endpoint with findings accumulated over one or more questions. hile this may severely affect differential diagnosis accuracy, it can be helpful for providing patients with summarized actions based solely on their first few discovered symptoms.
