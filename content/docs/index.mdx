## Core concepts

### Base URL

Base URL for all Avey APIs:

```http
https://api.avey.ai
```


### Authentication
Avey APIs requires an authentication token. The token must be included in the `X-API-KEY` header of each request:

```http
X-API-KEY: YOUR_API_KEY
```
