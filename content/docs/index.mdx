## Prerequisites

- **API Key (Bearer Token)** issued by Avey  
- An HTTP client (e.g., cURL, Postman, or a server-side HTTP library)  

---

## Core concepts

### Base URL

Base URL for all Avey APIs:

```http
https://api.avey.ai
```


### Authentication
Avey APIs require Bearer token authentication. The token must be included in the `Authorization` header of each request:

```javascript
Authorization: Bearer YOUR_API_KEY
```
